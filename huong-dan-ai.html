Cấu hình AI trong CodeLab

1
Mở Teacher Panel → ⚙️ Cấu hình
Nhấn icon ⚙️ trong sidebar hoặc menu giáo viên. Chọn tab 🤖 AI API.
2
Chọn provider và paste API key
Nhấn vào card của provider muốn dùng (Claude / ChatGPT / Gemini).
Paste API key vào ô API Key, nhấn Lưu.
✅ Key hiển thị dạng sk-ant-api03-••••3abc sau khi lưu — bình thường, đây là ẩn bảo mật.

Sinh bài tập

1
Mở tab 🤖 AI Sinh bài tập
Trong sidebar giáo viên, nhấn 🤖 AI Sinh bài tập.
2
Paste nội dung bài học
Copy nội dung từ SGK PDF, slide, hoặc gõ tóm tắt vào ô văn bản. Không cần format đẹp — AI hiểu text thô.
Ví dụ nội dung tốt
Bài 17: Biến và lệnh gán

# Khái niệm:
Biến là tên đại diện cho giá trị trong bộ nhớ.
Python tự nhận kiểu: x=10 (int), s='An' (str), d=8.5 (float)

# Lệnh gán:
x = 10 # gán giá trị
a, b = 1, 2 # gán đồng thời
a, b = b, a # hoán vị
💡 Ít nhất 50 từ để AI có đủ context. Càng chi tiết, câu hỏi càng chính xác.
3
Điền cấu hình
Lớp K10 Python / K11 SQL / K12 HTML…
Tên bài VD: "Bài 17: Biến và lệnh gán"
Mô hình AI Haiku (nhanh) hoặc Sonnet (tốt hơn)
Số câu/mức 1 câu × 6 mức = 6 câu | 2 × 6 = 12 | 3 × 6 = 18
4
Chọn mức Bloom cần sinh
Tick vào các mức B1–B6. Mặc định B1–B4. Bỏ B5-B6 nếu muốn bài cơ bản hơn.
ℹ️ Mỗi mức Bloom = 1 lần gọi API. Chọn 4 mức = 4 lần gọi, chọn 6 mức = 6 lần gọi.
5
Nhấn ✨ Sinh bài tập với AI
Đợi 5–15 giây. Progress bar hiển thị từng mức Bloom đang xử lý. Kết quả xuất hiện ngay khi xong từng mức.

Review & Lưu vào ngân hàng

Sau khi AI sinh xong, mỗi câu hỏi hiển thị dạng card. Có 3 nút hành động:

👁
Xem đề — preview nội dung câu hỏi
Mở rộng card, hiển thị đề bài HTML và code mẫu. Kiểm tra xem câu hỏi có phù hợp không.
▶
Chạy thử — validate code mẫu bằng PyInterp
Chạy code mẫu ngay trong browser (không cần server). Kiểm tra output có đúng không.
✅ Nếu thấy ✅ Output khớp — câu hỏi có test case hợp lệ.
⚠️ Nếu output khác — hệ thống tự cập nhật expected_output theo output thực tế.
✕
Bỏ — xóa câu không dùng
Câu hỏi không phù hợp → nhấn ✕ để loại ra khỏi batch trước khi lưu.
💾
Sau khi review xong, nhấn 💾 Lưu vào ngân hàng. Hệ thống lưu đồng thời:
• Đề bài HTML → Sheets NoiDung
• Code mẫu → Sheets CodeMau
• Metadata (id, lớp, Bloom) → Sheets BaiTap

Tùy chỉnh Prompt (nâng cao)

Hệ thống dùng kiến trúc 2 tầng. GV chỉ cần quan tâm tầng 2.

Tầng 1 — System Prompt (ẩn, tự động)
└─ Context CT GDPT 2018, JSON schema output

Tầng 2 — User Template (GV chỉnh trong ⚙️ Cấu hình → 📝 Prompt AI)
└─ Biến: {{bloom}} {{bloom_desc}} {{grade}} {{chapter}}
{{content}} {{count}} {{examples}}

Few-shot tự động (thay thế {{examples}})
└─ Chọn 2 bài cùng type+Bloom từ ngân hàng 971 bài
AI học phong cách viết, không copy nội dung
1
Vào ⚙️ Cấu hình → tab 📝 Prompt AI
Chọn loại bài (Python / HTML / SQL) rồi chỉnh template trong ô editor.
2
Dùng nút Preview để kiểm tra
Nhập 1-2 câu nội dung mẫu → nhấn 🔍 Preview → xem đúng prompt sẽ gửi đến AI.

Gợi ý tùy chỉnh hay: Thêm dòng "Bài tập phải liên quan đến quản lý học sinh / thư viện / trường học" vào template để AI sinh bài sát thực tế hơn với học sinh Thủ Thiêm.

Câu hỏi thường gặp

AI sinh bài xong, tôi có thể sửa không?▾
Có. Sau khi lưu vào ngân hàng, mở 📝 Ngân hàng bài → chọn bài → nhấn ✏️ Chỉnh sửa. Đề bài, code mẫu, lý thuyết đều chỉnh được bằng RichText editor.
Code mẫu AI sinh ra bị sai, làm sao sửa?▾
Nhấn ▶ Chạy thử trước khi lưu. Nếu output sai, hệ thống hiển thị expected vs actual — có thể chấp nhận output thực tế làm test case. Sau khi lưu, dùng editor trong Ngân hàng bài để sửa code mẫu.
Lỗi "Invalid API key" hoặc "Unauthorized"?▾
Kiểm tra: (1) key copy đúng chưa, không có dấu cách thừa; (2) tài khoản có credit chưa (vào platform.claude.com → Billing); (3) key chưa bị revoke. Thử nút 🧪 Test AI để kiểm tra.
Bài AI sinh có lưu vào Google Sheets không?▾
Chỉ lưu khi bạn nhấn 💾 Lưu vào ngân hàng. AI sinh bài xong chỉ hiển thị trong browser, chưa lưu đâu cả. Bạn review xong mới quyết định lưu câu nào.
Có thể sinh bài cho nhiều bài SGK cùng lúc không?▾
Không, mỗi lần chỉ 1 bài. Nhưng mỗi lần sinh chỉ mất 10–15 giây và ~$0.02. Để sinh 30 bài SGK K10: 30 × 15s = 7.5 phút, chi phí ~$0.60 (15,000đ).
Tôi muốn AI sinh theo phong cách đề thi THPT, làm sao?▾
Vào ⚙️ Cấu hình → 📝 Prompt AI → chỉnh User Template, thêm dòng: "Câu hỏi theo chuẩn đề thi THPT Quốc gia, có mã đề, thang điểm rõ ràng". Nhấn 💾 Lưu. Lần sinh tiếp theo sẽ theo phong cách đó.
Gemini miễn phí bị giới hạn rate, có cách nào bypass không?▾
Không cần bypass. Giới hạn 15 req/phút đủ dùng: sinh 6 mức Bloom = 6 requests, mất 24 giây. Nếu gặp lỗi 429 (rate limit), đợi 1 phút rồi thử lại. Hoặc chuyển sang Claude/OpenAI với chi phí rất nhỏ.
